# 새로운 파일 선택 방법 테스트 안내

FilePicker LateInitializationError 문제를 해결하기 위해 새로운 파일 선택 방법을 구현했습니다.

## 변경 사항
1. FilePicker 패키지 의존성 완전 제거
2. 수동 파일 경로 입력 방식으로 변경
3. 파일 경로 도움말 기능 추가

## 테스트 방법

### 1. Excel 파일 준비
```
상품명    | 바코드
---------|----------
담배A     | 1234567890123
담배B     | 2345678901234
담배C     | 3456789012345
```

### 2. 파일 경로 확인하기
1. 안드로이드에서 파일 관리자 앱 열기
2. Excel 파일 찾기 (보통 Downloads 폴더)
3. 파일을 길게 누르고 "속성" 또는 "정보" 선택
4. 전체 경로 복사 (예: `/storage/emulated/0/Download/inventory.xlsx`)

### 3. 앱에서 테스트
1. 앱 실행
2. 파일 경로 입력 필드에 전체 경로 입력
3. "경로에서 불러오기" 버튼 클릭
4. Excel 파일이 정상적으로 로드되는지 확인

## 예상되는 장점
- FilePicker LateInitializationError 완전 해결
- 안정적인 파일 접근
- 더 직접적인 파일 선택 방법
- 파일 권한 문제 우회

## 만약 여전히 문제가 있다면
"파일을 찾을 수 없습니다" 오류가 발생하면:
1. 파일 경로가 정확한지 확인
2. 파일이 실제로 해당 위치에 있는지 확인
3. 앱에 파일 읽기 권한이 있는지 확인
